HexDrop — Quick Reference
LOADING A FILE
DropDrag any file onto the drop zone to open it
BrowseClick the drop zone to pick a file from your computer
Load new fileOnce a file is open, drag a new file anywhere onto this window — it replaces the current file immediately. You do NOT need to close or reopen HexDrop.
Popup dropDrop a file directly on the HexDrop popup (toolbar icon) — it opens the viewer automatically in this tab
FIND VALUE BAR — searches by what a byte CONTAINS (Ctrl+F)
HEX modeType hex bytes separated by spaces — e.g. 4D 5A finds all MZ headers. Or type plain ASCII text.
DEC modeType decimal byte values 0–255 — e.g. 128 finds every byte with value 128 (same as hex 80). Multiple: 77 90
▼ NEXT / ▲ PREVStep through every match in the file (also Enter / Shift+Enter)
✕ CLEARClear search and remove all highlights (also Escape)
TOOLBAR — Jump to Address · Encoding · Export
Jump to AddressType a byte ADDRESS (position in file) and press ↵ — scrolls directly to that byte and selects it
HEX / DEC toggleNext to Jump — switches between hex address (e.g. 3C) and decimal address (e.g. 60)
EncodingChanges how the ASCII column decodes bytes — UTF-8, UTF-16 LE/BE, Windows-1252, ASCII
ColsSets how many bytes appear per row — 8, 16, or 32
EXPORT ▾ — three formats to download or share
Hex DumpDownloads the full file as a formatted grid: address · hex bytes · ASCII. The standard format — paste into forum posts, bug reports, or share with other reverse engineers.
Hex OnlyDownloads raw bytes as hex text, 16 bytes per line (e.g. 4D 5A 90 00). Paste directly into Python scripts, Cheat Engine, or patch tools.
Decoded TextDownloads the file decoded as readable text using the current encoding. Perfect for sharing game strings from a UTF-16 LE save — the recipient only needs Notepad to read it.
Save as PDFOpens a print-ready hex dump in a new tab and triggers Chrome's print dialog — choose Save as PDF as the destination. Great for archiving or sharing a formatted document.
HEX VIEWER (main area)
Click a byteSelects it — STATUS bar shows its Hex value, character, and decimal number
Click + dragSelects a byte range — Copy Hex or Copy Text buttons appear in the STATUS bar
Page Up / DownNavigate through large files one page at a time (16 KB per page)
STATUS BAR (bottom bar)
Offset: 0x3C (60)The byte's ADDRESS — where it lives in the file. The (60) is the decimal address to type into Jump to Address in DEC mode.
Hex / Char / DecThe byte's VALUE — what is stored there in three formats. Note: this Dec is the VALUE (e.g. 128), not the address.
Copy HexCopies selected bytes as a hex string (e.g. 4D 5A 90 00)
Copy TextCopies selected bytes decoded as text in the current encoding
PAGECurrent page and total — use ◀ PREV / NEXT ▶ to navigate large files
